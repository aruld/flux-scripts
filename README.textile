h1. Flux Scripts for managing workflows

h2. Upload workflow files to Flux repository:

@./upload-workflow.sh template.ffc template@

h2. Download workflow files from Flux repository:

@./download-workflow.sh template@

h2. Deploy workflow to Flux engine:

@./deploy-workflow.sh /template template.ffc@

h2. Remove workflow from Flux engine:

@./kill-workflow.sh /template@
